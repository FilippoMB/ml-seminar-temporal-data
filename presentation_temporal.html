Motivation

Temporal data is about usable memory

The signal is often not in one observation, but in how observations change.

  • Horizon: what future or label?
  • Order: which past evidence matters?
  • Missingness: gaps, clouds, irregular dates
  • Leakage: keep future information hidden

Past context is useful only relative to a prediction horizon.

Nonlinear windows

MLP: a trainable, nonlinear window operator

  • Task: Learn the map \(X_{t-W:t} \mapsto \widehat{X}_{t:t+H}\)
  • Model: hidden layers learn nonlinear relationships among lags and variables
  • Batching: many window-target pairs are stacked and processed in parallel
Limitation

What should \( H \) be?

  • Too short: miss evidence
  • Too long: add noise
  • Memory becomes hand-designed

Reservoir computing

An Echo State Network trains only the readout

  • Random recurrent dynamics
  • Rich temporal features
  • Train only the readout
\[ \begin{aligned} h_t&=\tanh(W_i x_t + W_h h_{t-1})\\ \hat{y}_{t+\tau}&=W_o h_t \end{aligned} \]
Reservoir practice

Useful when training recurrence is expensive

  • Spectral radius: memory
  • Input scaling: nonlinearity
  • Ridge readout: overfitting control

Useful when recurrent training is too expensive.

Temporal convolutional networks

Depth and dilation expand the receptive field

  • Local filters compose
  • Dilation skips positions
  • Depth grows context
