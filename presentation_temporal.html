Motivation

Temporal data asks for memory

We observe a process through ordered measurements and want a model that keeps the useful past while ignoring irrelevant variation.

  • Forecast: predict future values.
  • Classify: assign a label to a whole sequence.
  • Detect: find changes, events, or anomalies.
  • Impute: reconstruct missing observations.

A sequence model maps past context and covariates to future or sequence-level outputs.

Earth observation

Remote sensing turns places into sequences

A pixel, parcel, sensor footprint, or region can be represented as a temporal signal after spatial and spectral preprocessing.

  • Vegetation indices trace phenology across the growing season.
  • Clouds, snow, and orbit schedules create gaps and irregular dates.
  • Labels may describe crops, land cover, hazards, or change events.

Formal setup

A minimal sequence-learning view

\[ \mathcal{F}_{\theta}\left(x_{t-W:t}, u_{t-W:t+H}\right) = \hat{y}_{t:t+H} \]
  • \(x_t\): observed signal, possibly multivariate.
  • \(u_t\): exogenous context such as calendar, weather, orbit, mask, or metadata.
  • \(W\): context length. \(H\): output horizon.

Learning across collections

Local and global predictors

Local models

One model per signal: tailored, but data inefficient.

\(\hat{y}^i_{t+h}=f(x^i_{t-W:t};\theta^i)\)

Global models

One shared model: sample efficient and scalable across many related series.

\(\hat{y}^i_{t+h}=f(x^i_{t-W:t}, a^i;\theta)\)

Remote-sensing datasets usually favor global models because many pixels or parcels share seasonal structure.

Fixed windows

A sliding context becomes a table

Choose \(P\) past values and learn a map from lagged features to the target.

\[ \hat{y}_{t+\tau}=g\left([x_{t-P},\ldots,x_{t-1},x_t]\right) \]
  • Linear AR: \(g\) is linear.
  • Tree, SVM, or MLP: \(g\) can be nonlinear.
  • Every new time step shifts the design row.
Nonlinear windows

An MLP is a learned window operator

  • Input dimension: \(d_{\mathrm{in}}=P \times\) variables.
  • Output dimension: one horizon, multiple horizons, or a class label.
  • Nonlinear layers learn interactions among lags and covariates.
  • Training pairs come from shifting long sequences into many examples.
Baseline example

Electricity demand: nonlinear and seasonal

The load-temperature relation is often U-shaped rather than linear.

  • Demand depends on temperature, calendar, and daily or weekly rhythms.
  • Seasonal persistence is a strong baseline: tomorrow often resembles yesterday.
  • Neural models must beat simple temporal baselines before we trust the extra complexity.

This same discipline matters for Earth observation: compare to phenology curves, seasonal medians, and simple change rules.

Limitation

What should \(P\) be?

  • Too short: relevant evidence is outside the window.
  • Too long: more parameters, more noise, weaker local resolution.
  • Fixed windows make memory an input-design decision.

The rest of the lecture asks how architectures learn, share, or select memory.

RNN idea

A hidden state carries context

  • Process one observation at a time.
  • Update a hidden state \(h_t\) with the new input and old state.
  • Use \(h_t\) as a compressed summary of the past.
\[ h_t=\phi(W_x x_t + W_h h_{t-1}), \qquad y_t=W_o h_t \]

The same recurrent update is reused at every time step.

Training

Backpropagation through time

  • Unroll the recurrence into a deep feed-forward graph.
  • Share parameters across all time steps.
  • Gradients travel through many repeated transformations.

Long memory is possible, but optimization and sequential compute can become the bottleneck.

Gated recurrence

LSTMs and GRUs learn what to keep

Write

How much new evidence should enter the state?

Forget

Which old information should decay or be overwritten?

Read

Which part of memory is useful for the current output?

For crop mapping, this can be read as a latent growth-stage tracker: emergence, peak greenness, senescence, harvest.

Reservoir computing

An Echo State Network trains only the readout

  • Random input and recurrent weights create a high-dimensional dynamical system.
  • The reservoir state \(h_t\) contains many nonlinear temporal features.
  • Only the output map is fitted, often with ridge regression.
\[ h_t=\tanh(W_i x_t + W_h h_{t-1}), \qquad \hat{y}_{t+\tau}=W_o h_t \]
Reservoir practice

Useful when training recurrence is expensive

  • Spectral radius controls contraction and memory.
  • Input scaling controls the linear or nonlinear operating regime.
  • Readout regularization controls overfitting when labels are limited.

Reservoirs trade supervised recurrent training for a rich, fixed pool of dynamical features.

Convolution

A filter is a sliding learned window

A 1-D convolution applies the same small kernel at every time position.

\[ z_t=\sum_{k=0}^{K-1} w_k x_{t-k} \]
  • Like AR: local lagged features.
  • Unlike AR: many nonlinear channels stacked in depth.
  • Unlike RNN: all positions can be processed in parallel.
Temporal convolutional networks

Depth and dilation expand the receptive field

  • Causal padding prevents future leakage.
  • Dilation skips positions to cover longer history.
  • Residual blocks make deep temporal filters trainable.

A TCN keeps the window intuition while escaping one fixed, shallow window size.

Remote-sensing fit

Temporal filters detect local motifs

Classic window model

  • Explicit lag vector.
  • One predictor sees the full chosen context.
  • The window length is an input-design decision.

TCN model

  • Small learned filters scan the season.
  • Depth composes emergence, peak, and decline patterns.
  • The receptive field grows by architecture.

TCNs have been used directly for satellite image time-series classification, e.g. Pelletier, Webb, and Petitjean, 2019.

Self-attention

Content-addressable memory over time

\[ \mathrm{Attn}(Q,K,V) =\mathrm{softmax}\left(\frac{QK^\top}{\sqrt{d}}\right)V \]
  • Each time step produces a query, key, and value.
  • Attention weights decide which other steps matter.
  • Multi-head attention learns several relation types at once.
Transformers

Why attention changed sequence modeling

Strengths

  • Direct long-range interactions.
  • High parallelism during training.
  • Flexible conditioning on dates, masks, metadata, and covariates.

Costs

  • Quadratic attention in sequence length.
  • Less built-in locality than convolutions.
  • Data hungry without good inductive bias.

Transformer architecture: Vaswani et al., 2017.

Satellite sequences

Attention can use dates, masks, and metadata

Satellite time series rarely arrive as clean daily grids. The model often needs to know what was observed and when.

  • Acquisition date can be encoded as day-of-year or positional features.
  • Cloud and quality masks prevent missing pixels from pretending to be signal.
  • Metadata such as sensor, orbit, or region can condition the sequence encoder.

State space models

Continuous-time intuition, neural sequence model

\[ \dot{s}(t)=A s(t)+B u(t), \qquad y(t)=C s(t)+D u(t) \]
  • A latent state \(s(t)\) evolves under linear dynamics.
  • Inputs drive the system, outputs read from the state.
  • After discretization, the model becomes a recurrence or a long convolution kernel.

Why this lands for physics and CS

SSMs connect control-theory dynamics with trainable neural blocks for long sequences.

S4 and Mamba

State updates can be scanned efficiently

  • S4 parameterizes SSMs so very long dependencies become practical.
  • Mamba makes SSM parameters input-dependent, so the model can selectively keep or forget information.
  • The practical promise: RNN-like streaming with long-context efficiency closer to convolution or scan operations.

S4: Gu et al., 2021. Mamba: Gu and Dao, 2023.

Architecture map

Which memory mechanism fits which problem?

Architecture Memory mechanism Parallelism Good fit Watch out
Window + MLP Fixed explicit lags High Baselines, small tabular forecasting, engineered EO features Choosing \(P\); weak long memory
RNN/GRU/LSTM Learned recurrent state Low across time Streaming data, compact encoders, latent stage tracking Optimization and long sequences
Reservoir/ESN Random dynamical features Medium Fast training, limited labels, readout experiments Hyperparameter sensitivity
TCN Receptive field by depth/dilation High Phenology, dense sensor streams, local motifs Very long context needs depth or dilation
Transformer Direct attention over observed dates High Irregular observations, masks, multimodal covariates Data and memory cost
SSM/Mamba State space scan with selective memory High in training, streaming in inference Long EO sequences, efficient context, physics-friendly framing Newer ecosystem; less standard for many EO workflows
Takeaways

Same question, different memory mechanisms

State

RNNs and reservoirs compress the past into a moving representation.

Filter

Temporal convolutions learn local windows and compose them into large receptive fields.

Select

Attention and selective SSMs decide which time steps or state components matter.

For temporal data, architecture is a hypothesis about what kind of past matters.
